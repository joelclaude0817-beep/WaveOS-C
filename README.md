# WaveOS
Hello, Let me show you how i made a linux-like operating system called "WaveOS".

To Clone The Repository:
```bash
git clone https://github.com/joelclaude0817-beep/WaveOS-C/
```

To change directorys:
```bash
cd WaveOS-C
```

To change mods:
```bash
chmod +x *
```

To run:
```bash
gcc -o waveos waveos.c
./waveos
```

And **You** just got my operating system made by me.
